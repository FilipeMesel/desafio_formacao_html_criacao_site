O que é o FreeRTOS?

voltar

O FreeRTOS é um sistema operacional em tempo real de código aberto que é amplamente utilizado em sistemas embarcados. Ele permite que você divida seu código em tarefas independentes e gerencie recursos compartilhados de forma eficiente.

Por que o ESP32?

voltar
  1. O ESP32 é um microcontrolador poderoso e acessível
  2. Combina conectividade Wi-Fi e Bluetooth.
  3. É ideal para projetos IoT, automação residencial, robótica e muito mais.
